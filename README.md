# Plant Builder

Procedural plant generation system developed for The Pond project.

This package provides a runtime system, shaders, and assets to assemble and render
procedural plants using sprite atlases.

---

## Context

Plant Builder was developed specifically for The Pond project to support
procedural vegetation generation.

It is designed as an internal tool and is not intended to be a generic,
plug-and-play solution.

---

## Authors & Credits

- **Original development**: Augustin Grassien

---

## Package Content

- Runtime scripts for plant generation
- Custom HLSL shaders
- Sprite atlases used by the system
- Optional editor tools
- Sample scenes demonstrating usage
